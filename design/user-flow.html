Approval Flow — Pengadaan & Biaya

sequenceDiagram
    actor Pengaju as Pengaju (Admin/Staff)
    actor Mengetahui as Mengetahui (SPV/Manager)
    actor Menyetujui as Menyetujui (Keuangan/Direktur)
    participant System as ERP System

    Pengaju->>System: Submit Pengajuan PR / Biaya
    System->>Mengetahui: Notifikasi Approval
    Mengetahui->>System: Approve / Reject
    alt Rejected
        System->>Pengaju: Pengajuan Ditolak
    else Approved
        System->>Menyetujui: Notifikasi Approval
        Menyetujui->>System: Approve / Reject
        alt Rejected
            System->>Pengaju: Pengajuan Ditolak
        else Approved
            System->>Pengaju: Pengajuan Disetujui
            System->>System: Masuk ke Tahap Berikutnya (PO/BAP/Bayar)
        end
    end
